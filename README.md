# agrinho
projeto realizado para o agrinho 
🎮 Mecânica principal
Trator: representado por um quadrado vermelho, pode se mover em todas as direções usando as setas do teclado.

Plantações: aparecem como círculos verdes espalhados pelo mapa. Ao coletá-las, o jogador ganha pontos, aumenta o número de colheitas e consome combustível.

Obstáculos: blocos cinzas que reduzem pontos e combustível ao serem atingidos.

⏱️ Recursos e desafios
Combustível: começa em 100 e vai diminuindo conforme o jogador colhe plantações ou bate em obstáculos.

Tempo: limitado a 90 segundos, criando uma pressão constante para colher o máximo possível.

Pontuação: cada plantação rende +10 pontos, cada obstáculo tira -5 pontos.

🌾 Objetivo
O objetivo é colher o maior número de plantações possível antes que o combustível ou o tempo acabem, evitando obstáculos para não perder pontos.

Esse estilo de jogo mistura agilidade, estratégia e gerenciamento de recursos, lembrando minigames clássicos de coleta e sobrevivência.
